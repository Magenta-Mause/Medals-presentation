## Getting Started

1. **Clone the repository:**

   ```bash
   git clone https://github.com/Magenta-Mause/Medals-presentation.git
   cd Medals-presentation
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Run the presentation:**
   ```bash
   npm start
   ```

## Customization

- Edit the presentation content in the `src/` directory.
- Refer to the [Motion Canvas documentation](https://motioncanvas.io/docs) for advanced features.
